
# CPDLC System

Simple Controller Pilot Data Link Communication system.

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cpdlc-system.git
   cd cpdlc-system
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## Requirements
- Node.js
- MongoDB
